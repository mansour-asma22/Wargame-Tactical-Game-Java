Wargame – Tactical Strategy Game (Java)

Un jeu de stratégie tactique développé en Java, mettant en scène deux joueurs qui s’affrontent sur un plateau. Le projet met l’accent sur la logique de jeu, la gestion des unités, l’influence du terrain et la conception d’une architecture logicielle maintenable.

* Présentation du projet

Wargame est un projet académique visant à implémenter un jeu au tour par tour dans lequel chaque joueur contrôle plusieurs unités disposant de caractéristiques propres (points de vie, attaque, déplacement…).
Le terrain, découpé en grille, influence les déplacements et les combats, ajoutant une dimension stratégique.
Ce projet a permis de consolider les compétences en programmation orientée objet, conception logicielle, organisation du code et gestion de projet avec Git.

* Fonctionnalités

Déplacement d’unités sur une grille 2D
Système de combat : attaque, défense, points de vie
Influence du terrain (zones bloquées, cases spéciales…)
Gestion des tours pour chaque joueur
Validation des actions (portée, disponibilité, règles du jeu)
Architecture modulaire, permettant d’ajouter facilement de nouvelles unités ou règles

* Technologies et outils

Java
Programmation Orientée Objet (POO)
IDE : Eclipse 
Gestion de la logique du jeu via structures de données 2D

* Lancer le projet

Compilation : javac Main.java
Exécution :java Main

*Objectifs pédagogiques

Conception d’un système de jeu complet en Java
Application avancée de la POO
Utilisation d’un plateau 2D et de mécanismes de combat
Gestion de la cohérence des actions (déplacements, attaques…)
Structuration propre d’un projet logiciel
